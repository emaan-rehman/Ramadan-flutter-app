# Ramadan Flutter App

## Overview

A Flutter application designed to help Muslims during Ramadan by providing useful Islamic tools in a single mobile application.

## Features

- Ramadan Timings
- Daily Duas
- Digital Tasbeeh Counter
- Habit Tracker
- Zakat Calculator
- User-Friendly Interface

## Technologies Used

- Flutter
- Dart
- Material Design
